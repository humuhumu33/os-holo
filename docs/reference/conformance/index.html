Conformance rows

A conformance row is one claim about the OS, paired with the witness that proves it. node tools/gate.mjs re-runs every witness, joins the results to this catalog, and exits non-zero if any required row is unproven — so a non-conforming build cannot ship.

Note

This page is generated from system/os/etc/conformance.jsonld. 112 rows · 112 required.
